## Overview<a name="overview"></a>
Hydro3DNet proposes a unified Transformer framework that fuses LiDAR point clouds with hydrological time-series to achieve real-time 3D object detection in inland-water scenarios.  
This repository hosts supplementary materials; **the `src/` directory will appear once the review process is complete.**
